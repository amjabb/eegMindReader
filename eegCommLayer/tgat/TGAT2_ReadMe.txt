First, install the SPIDEV python package found here:
https://github.com/doceme/py-spidev
Also pip install Raspberry Pi 3B+ RPi GPIO modules

The TGAT Breakout modules are connected and communicated with over SPI.
 
Once the interconnections between the TGAT Breakout modules are completed, run get_data.py in order to output EEG data into *.csv files.
